# TiendaMS

Tienda web desarrollada con PHP y MySQL. Sistema funcional con panel administrativo para gestión de productos.

## Tecnologías
- PHP
- MySQL  
- JavaScript
- HTML5 / CSS3

## Funcionalidades
- Visualización de productos en tienda
- Panel administrativo (CRUD de productos)
- Interfaz responsive y dinámica

## Demo en vivo
[Ver tienda funcionando](https://tiendams.store/) 

## Instalación local
1. Clonar repositorio
2. Importar base de datos (archivo database.sql)
3. Configurar conexión en `config/db.php`
4. Ejecutar con XAMPP / WAMP
